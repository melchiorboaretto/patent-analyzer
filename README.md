# Patent Analyzer

This is a simple vacation project
to build an SQL-based (or something alike) system.

## Why?

Redundantly, because I found it a cool challenge
to test my skills while on vacation from college.

## For what?

It is meant to be used as an advanced historic data viewer: it is 
nice to see how these things changed the world, in which pace, etc...


The idea is to build something complete and
simple to use. I might build a TUI (inspired by Lazygit) to 
make it cooler to look and use.


## Data source

I'm using USPTO PatentsView API to get data from.
